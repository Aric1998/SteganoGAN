============
Installation
============


Stable release
--------------

To install Deep Steganography, run this command in your terminal:

.. code-block:: console

    $ pip install deepsteganography

This is the preferred method to install Deep Steganography, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From source
------------

The source for Deep Steganography can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/DAI-Lab/deepsteganography

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/DAI-Lab/deepsteganography/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ make install


.. _Github repo: https://github.com/DAI-Lab/deepsteganography
.. _tarball: https://github.com/DAI-Lab/deepsteganography/tarball/master
